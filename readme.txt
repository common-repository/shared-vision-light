== Description ==

Shared Vision is a 3rd-party service that helps Creators manage their affiliate links on their website. In order to use this plugin, you must have a Creator account, which you can register for at https://joinsharedvision.com. We have a free plan available, using this plugin does not require you to be a paying customer of Shared Vision. Once you created your first Shopping List on the Shared Vision platform, you can use this plugin to display the list on your Wordpress site. To learn more about Shared Vision, please see the following pages:

- https://joinsharedvision.com
- https://joinsharedvision.com/terms-and-conditions
- https://joinsharedvision.com/privacy-policy

The Shared Vision Light Wordpress plugin brings power of the Shared Vision platform to your site! We made this plugin specifically for older websites, where the Wordpress or PHP version are not current.

The plugin will place a Shared Vision script on your website, which allows displaying the Shopping Lists you created with Shared Vision. The plugin is calling our 3rd-party library, and is placing a script on your website that is required for rendering the Shared Vision lists you created on the Shared Vision platform.

Features:
 - Display previously created shopping lists
 - Shopping Lists are automatically updated, no need to edit your post

== Frequently Asked Questions ==

= How to I create a Shared Vision Account? =

Go to https://creator.joinsharedvision.com/register?ref=wordpress&plan=proYearly to signup for your free account and a 30day trial!

= How do I get my API Key? =

When logged in to your Shared Vision account, go to: Settings => Integrations to get your API key

= What happens to my old embedded lists? =

Nothing! Your old embedded lists will work just as they have before. This Plugin is the newest way you can add lists to your website.

== Installation ==

This section describes how to install the Shared Vision plugin

1) Install

	* Go to the WordPress Dashboard "Add New Plugin" section.
	* Search For "Shared Vision Light"
	* Install, then Activate it.
2) Configure

	* Go to the Settings-> SharedVision Page
	* Add your API Key
	* That's it, you are ready to manually embed lists from Shared Vision
